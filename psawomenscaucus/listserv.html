PSA Women's Caucus: Listserv

List archives

Directions for subscribing to the list

Send email to:
sympa@ucdavis.edu
with the following in the Subject of the message:
subscribe psawomenscaucus <first name> <last name>
(Leave the body of the message blank)

For example, if Jane User wanted to subscribe to the list, she would send a blank email with the following Subject to <sympa@ucdavis.edu>:
subscribe psawomenscaucus Jane User

Directions for unsubscribing from the list

Send a blank email to:
sympa@ucdavis.edu
with the following in the Subject of the message:
unsubscribe psawomenscaucus

Important: You need to send the "unsubscribe" request from the same email address that you used to subscribe to the list. If you have trouble unsubscribing, please contact Susan Hawthorne at schawthorne@stkate.edu.

Directions for sending email to the list

The email list for the PSA Women's Caucus is intended to be a low-traffic list, consisting of announcements relevant to the goals of the PSA Women's Caucus. However, if you have a posting that is relevant to the group, send it to: <psawomenscaucus@ucdavis.edu>

Important: Please do not use the address <psawomenscaucus@ucdavis.edu> for unsubscribing, since email to this address will go to everyone on the list. Instead, please follow the instructions for unsubscribing above.
